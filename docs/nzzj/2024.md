# 2024 年终总结

（日期：2024-12-31）

动笔写这篇文章的时候，正好也看到了 <https://www.munue.com/661.html> 上的“字述一年”的活动。我选的汉字是：“追”。

追，就是追随某个事物。今年我追的目标有3个：1、考上大学；2、建设网站；3、完成ICP备案。其中的两个已经完成；最后一个虽然完成，但是之后被取消了。原因可以看 <https://mcah.cc/blog-cancel-icp。>

2024年即将过去。就在2024年仅剩19天的日子，总结一下今年吧。

## 一、网站累计建设15个月，博客网站建设9个月

### （一）建设网站，必加博客组织

其实早在2023年9月开始，我就开始建设网站了。但是当时网站也没搞出什么太大的动静，导致互联网档案馆（<https://web.archive.org）都没能收录我当时的网站。>

2024年3月，我的博客网站开始建设。当时的我，玩互联网尚浅，还不懂如何去建设一个网站。后来我发现，博客网站如果要有流量，就必须要加入博客组织。于是，我陆续加入了开往等组织，才使得我的网站有人看。

### （二）建设网站，必要域名

域名有各种各样的类型。刚开始选购域名的时候，我就看到了 com 域名后缀。后来看到了上海的顶级域名后缀（.sh.cn），也是立刻买下了它。

但是随着我使用 .sh.cn 之后，我就发现，网站域名太长了，导致新访客难以记住网址。于是，我又买了 <https://mcah.cc> 域名，彻底解决了这个问题。

如果使用 Github Pages 自带的域名，或者用 .eu.org 域名，可能无法加入一些博客组织，这也是需要注意的。

### （三）建设网站，必要耐心

目前，我的博客网站已经建设了9个月，但是访客还是很少。好几次，我都想把网站关掉，一关了之。但是想到，建设网站是我一直以来的梦想，这也让我一直坚持下去。

其实，不论是做网站，还是治学，都是需要“耐心”这个东西。没有耐心，永远都无法坚持做事情。

无悔做网站！

## 二、考入大学

9月12日，进入了大专。这就代表着，我和之前的中学生涯彻底道别了。

我考入的大专是艺术类大专。语文、数学等学科也随之结束了。

在这三个月的大学生涯，我发现高中与大学有以下三改：

### （一）性格之改

高中时，我什么都看不上，但好在我会计算机，能帮助同学修复问题。高三，我被三个同学制裁，终于明白了这种性格是万万不可的。于是到了大学，终于改正了这个问题。

“看今日之他人，犹如看昨日之自己。”这是我花了好几年，才明白的道理。

### （二）学习方式之改

高中时期，老师教什么，我们学什么。大学不一样了。大学时期，什么都要看，老师所教的，远远不够。

我的计算机知识，主要都是在 <https://learn.microsoft.com> 上学的，主要就是有这个兴趣。我能在班里混出一个一席之地，主要就是靠着这个。

### （三）住宿之改

此前我是从来没有住宿过的。唯一一次在外住宿，就是2023年4月17日至21日在外面学农。所以9月12日晚上，我在博客网站上说：“月是故乡明。”

宿舍里3个河南省的，只有我一个上海本地人。一开始及其不适应，但是后来还是克服了 home-sick。住宿3个月，虽然每周都要回家，但是也逐渐能够实现家和学校的无缝衔接。

## 三、个人改变及所思

我经常说，搞一个博客网站，能大幅提高自己的写作能力。搞网站9个月，写作能力终于得到了提升。

再学编程语言，再看网络之知识。CIDR、TCP/IP，IPv4，这些如果不研究计算机就永远不可能懂的知识，我也终于能说出一二了。

之前有老师说，看到你玩计算机挺厉害的，一定要坚持啊。这东西不需要坚持，这就是我的爱好。

有些同学现在已经开始谈恋爱，处对象了。我对此不感兴趣。我认为，爱好就是对象。

箫鼓追随春社近，衣冠简朴古风存。古诗词告诉我们，人生需要豁达，乐观。

2024年，我乐观走过；2025，希望依旧。

再见，回忆满满的2024。欢迎希望满满的2025。
